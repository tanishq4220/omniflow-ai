🤖 Multi-Agent System Monitor
CrowdAgent Active
Calculates CDI = (people_count / max_capacity) × 100. Monitors spatial density thresholds and triggers alerts when exceeding 75%.
Formula: CDI = (count / capacity) * 100
QueueAgent Active
Estimates wait times by modeling queue dynamics. Triggers staff reallocation when wait exceeds threshold.
Formula: wait = queue_length / service_rate
SafetyAgent Active
Evaluates risk thresholds. CDI <50 → LOW, 50–75 → MEDIUM, 75–90 → HIGH, >90 → CRITICAL.
Output: LOW / MEDIUM / HIGH / CRITICAL
EmotionAgent Active
Analyzes attendee sentiment scores (0–100) to compute experience optimization recommendations.
Formula: score = 0.4(100-wait) + 0.3(100-CDI) + 0.3(emo)
